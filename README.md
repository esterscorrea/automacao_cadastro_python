# Projeto de Automação de Cadastro de Produtos

Este projeto foi desenvolvido como parte do curso **Intensivão de Python** da [Hashtag Treinamentos](https://www.hashtagtreinamentos.com/). 

## 🎯 Objetivo
O objetivo principal foi criar um script de automação (RPA) capaz de ler uma base de dados e cadastrar produtos num sistema web de forma automática, simulando o comportamento humano.

## 🎓 Créditos e Referências
- **Curso:** Intensivão de Python
- **Instituição:** Hashtag Treinamentos
- **Instrutor:** Lira Hashtag

## 🛠️ Minhas Contribuições e Adaptações
Embora a base do projeto tenha sido fornecida pelo curso, realizei adaptações críticas para garantir a execução no meu ambiente de desenvolvimento:
- **Resolução de Tela:** Calibração manual das coordenadas de clique (`x=757, y=264`) para MacBook Pro.
- **Gestão de Hardware:** Implementação de tempos de espera (`time.sleep`) customizados para lidar com a latência de um Mac Pro 2012 com macOS Catalina.
- **Estabilidade:** Uso de comandos de teclado combinados com cliques para garantir o foco nos campos do formulário.
